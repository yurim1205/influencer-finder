# Influencer-Finder
유튜버 탐색과 비교를 돕는 키워드 기반 인플루언서 검색 서비스

[서비스 링크](https://influencer-finder-eight.vercel.app/)

## 🧩 MVP 기능
- 키워드 기반 유튜버 검색
- 유튜버 리스트업 (10~20명)
- 채널 정보 제공
  - 채널명
  - 구독자 수
  - 최근 5개 영상 평균 조회수
  - 채널 설명 1문장
- 구독자 수 / 평균 조회수 기준 정렬

## 🛠️ 기술 스택
- Next.js (App Router)
- TypeScript
- Tailwind CSS
